Physics Engine#

jc_sim_oqp.physics.dispersive_hamiltonian(wc: float, wa: float, g: float, a: Qobj, sm: list[Qobj] | Qobj) → Qobj[source]#

Construct the Dispersive Hamiltonian.

Approximates the system when detuning Delta = wa - wc is large (|Delta| >> g*sqrt(n)). Includes the ac-Stark shift and Lamb shift.

H_disp = wc * a^dag * a + sum_i [ (wa + chi)/2 * sigma_z^i + chi * a^dag * a * sigma_z^i ]

where chi = g^2 / Delta.

Parameters:

  • wc (float) – Cavity frequency.

  • wa (float) – Atom frequency.

  • g (float) – Coupling strength.

  • a (qutip.Qobj) – Cavity destruction operator.

  • sm (Qobj | list[Qobj]) – Atom lowering operator(s).

Returns:

Dispersive Hamiltonian.

Return type:

qutip.Qobj
